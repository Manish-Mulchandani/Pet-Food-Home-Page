# Happy Dog Bites Landing Page

Figma design to HTML/CSS conversion assignment.

## Project

Converted a Figma design file to a responsive HTML and CSS landing page.

## Implementation

- Pure HTML5 and CSS3 (no frameworks)
- All images and assets stored locally
- Clean, semantic HTML structure


## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # Stylesheet
└── images/         # Local image assets
```

## Setup

Open `index.html` in a browser. No build process or dependencies required.
